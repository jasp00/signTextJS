Working with signatures

Decode Base64 string

base64 -d signature.b64 > signature.der

Verify signature

With Network Security Service tools:

signver -d dbdir -i content -s signature.der

With OpenSSL utility:

openssl smime -verify -content content -in signature.der -inform der -CAfile CA.crt -out /dev/null

Dump signature structure

With Network Security Service tools:

derdump -i signature.der

With OpenSSL utility:

openssl cms -in signature.der -inform der -cmsout -print
